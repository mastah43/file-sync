# File Sync

Addresses the synchronization from Google Drive to Proton Drive to be able to use Google Drives great authoring tools 
but have the availability of Proton Drive for European users if one day Google decides to block access.
This shall provide a data privacy safe option for synchronization running locally compared to online services like [cloudlinker](https://app.cloudslinker.com/) which get full file access on cloud storages.

But wait, there [rclone](https://rclone.org) already ..
